# 📷 Sentry Deck

[![Release](https://img.shields.io/github/release/danielchalmers/SentryDeck?include_prereleases&style=flat-square)](https://github.com/danielchalmers/SentryDeck/releases)
[![License](https://img.shields.io/github/license/danielchalmers/SentryDeck?style=flat-square)](LICENSE)

Review Tesla dashcam and Sentry Mode footage on Windows.

![Sentry Deck](https://github.com/user-attachments/assets/5b2efe25-bac3-4902-8e0e-e006c92c5ddf)

## Features

- **One timeline** — clips grouped by day with thumbnails, event reason, and location. Search across clips, places, and events.
- **Jumps to the event** — opens each clip at the action instead of the top of the buffer, like the car does, landing a few seconds before the Sentry/Honk/braking moment so you catch the lead-up. A marker on the seek bar pins the exact instant; press **E** to snap to it.
- **Every angle** — watch a 2×2 grid or any single camera, with live previews of all four. Click a preview to switch.
- **Finds your footage** — auto-detects the `TeslaCam` folder on a connected USB drive, or point it at any folder.
- **Smooth playback** — variable speed (0.25×–16×), scrubbing, and previous/next clip navigation.
- **Cut & share** — hit **Trim**, mark the start and end, and export that section as a single MP4, or right-click a clip to save ±30s around its event. Lossless and instant — no re-encoding.

## Install

1. Download the latest `SentryDeck-x64.zip` (or `-arm64`) from the [Releases](https://github.com/danielchalmers/SentryDeck/releases) page. An `.msi` installer is also available.
2. Extract and run `SentryDeck.exe`.
3. On first launch it offers to download FFmpeg (~80 MB) if it isn't already present.

Requires Windows 10 or 11.
