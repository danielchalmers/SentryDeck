# 📷 Sentry Replay

[![Release](https://img.shields.io/github/release/danielchalmers/SentryReplay?include_prereleases&style=flat-square)](https://github.com/danielchalmers/SentryReplay/releases)
[![License](https://img.shields.io/github/license/danielchalmers/SentryReplay?style=flat-square)](LICENSE)

**Sentry Replay** is a modern desktop application designed to make viewing Tesla Dashcam and Sentry Mode footage effortless. No more juggling multiple video files—view all angles simultaneously in a seamless, unified interface.

## 🚀 Getting Started

### Installation
1. Download the latest version from the [Releases](https://github.com/danielchalmers/SentryReplay/releases) page.
2. Extract the ZIP file to a folder of your choice.
3. Run `SentryReplay.exe`.
4. On first launch, the app will offer to download **FFmpeg** automatically if it's not found.

## 📖 How to Use

1. **Connect your USB drive**: Plug in the drive containing your `TeslaCam` folder.
2. **Select Folder**: Point Sentry Replay to your `TeslaCam` directory.
3. **Browse Clips**: Use the sidebar to browse through Recent Clips, Sentry Clips, and Saved Clips.
4. **Playback**: Click a clip to start playback. Use the seek bar and speed controls to navigate.
5. **Export**: Use the "Render" feature to create a single, shareable video of an entire event.
